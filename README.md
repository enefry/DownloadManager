# DownloadManager 下载管理器

DownloadManager 下载管理器是一个功能强大的文件下载管理组件，支持单文件下载、批量下载、断点续传、分片下载等功能。

## 主要特性

### 1. 核心下载功能
- 单文件分块下载
- 多文件队列下载
- 网络状态监控
- 网络类型判断（WiFi/蜂窝网络）
- 自动重试机制

### 3. 存储管理
- 磁盘空间检查
- 临时文件管理
- 文件合并
- 自动清理机制

### 4. 错误处理
- 错误类型定义
- 重试机制
- 错误恢复建议
